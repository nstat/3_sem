Сравнить на практике 3 способа взаимодействия между двумя процессами.
Для этого:
1) Разработать 3 программы, передающие файл произвольного размера между двумя процессами с использованием следующих инструментов:
  а) Разделяемая память SYS V.
  б) Очереди сообщений SYS V.
  в) Разделяемая память "своими руками" - настройка отображения с помощью вызова mmap(...)
2) Для каждой из программы измерить время выполнения при передаче файлов различных размеров, используя утилиту time либо средства из заголовка time.h: https://ru.wikipedia.org/wiki/Time.h, проводя измерения для 2-3 характерных размеров буфера (памяти, данных в сообщении): "малый" размер - менее пары сотен байт, 4 Кб, большой размер - больше 4 Кб.
3) По результатам измерений построить 3 графика времени передачи от размера файла (срез для соответствующего размера буфера). На каждом графике будут расположены 3 кривых для инструментов 1а, 1б, 1в. Интерполировать графики (получать кривые из экспериментальных точек) можно любым известным способом. После чего следует сделать выводы и заключения.
